Get the Light (Python)

Get the light is a Python game utilising the PyGame module. The player plays as a moth which can be moved around the screen using the arrow keys. There are three birds and one light bouncing around the screen and the user's objective is to get the moth to the light. If you manage to reach the light, you win and the game ends. However, if a bird collides with you at any point, that's game over; you lose.

Be careful, though: the speed of all characters on the screen will increase the longer you play.
